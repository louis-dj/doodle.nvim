## Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
  "louis-dj/doodle.nvim",
  config = function()
    require("doodle").setup()
  end
}
```


### [packer.nvim](https://github.com/wbthomason/packer.nvim)

```lua
use({
  "louis-dj/doodle.nvim",
  config = function()
    require("doodle").setup()
  end
})
```
